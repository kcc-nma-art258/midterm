Media Queries

Media queries are used to allow the the framework to be fluid, adapting to different screen sizes. The breaking points for firing are 640px(mobile), 1024px(tablet) and 1440px(Cinema Display).

Utilites

There are utilities that are used on this framework as a means to assist you with keeping your code dry by not repeating yourself by using variables and mixins to replace repetitive code. We are also using SASS a preprocessor that helps to minify code for faster load time aswell as allowing you to break your code up by sections that makes editing a breeze. There is also a clearfix hack included that is used to clear floating child elements from their parent.
